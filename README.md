# MCU_Brick_PCB_Routing
Проект разводки печатной платы для микроконтроллерного модуля в рамках Международной инженерной школы. Моей задачей была трассировка платы и подготовка к производству.

Что было сделано:
* Трассировка печатной платы (PCB Layout) в Altium Designer.
* Учет правил проектирования (DRC).
* Плата была успешно произведена на заводе (см. фото).
